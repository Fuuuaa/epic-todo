# UpNext-ToDo

**UpNext** — современный список задач с аккуратным дизайном и плавными эффектами.
Проект демонстрирует работу с состояниями, списками и взаимодействием компонентов в React.

## Функционал

- Добавление новой задачи
- Редактирование задачи
- Удаление задачи
- Отметка выполненных
- Счётчик активных задач
- Удаление всех задач
- Плавные визуальные переходы

## Стек

- React (Vite) — логика и UI
- CSS — стилизация

## Установка и запуск

```bash
npm install
npm run dev
```
